# Employee Sentiment Analysis

## Overview
This project analyzes employee messages to identify sentiment, engagement, and flight risk using NLP and statistical analysis.

## Tasks
1. Sentiment Labeling (Positive, Negative, Neutral)
2. Exploratory Data Analysis (EDA)
3. Monthly Sentiment Scoring
4. Employee Ranking
5. Flight Risk Identification
6. Predictive Modeling (Linear Regression)

## Results
- **Top 3 Positive Employees:** See `outputs/top3_positive_2011_12.csv`
- **Top 3 Negative Employees:** See `outputs/top3_negative_2011_12.csv`
- **Flight Risks:** `outputs/flight_risks.csv`
- **Regression Performance:** See `outputs/regression_performance.png`
